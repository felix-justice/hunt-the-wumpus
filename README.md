# Hunt the Wumpus

A Python recreation of the classic Hunt the Wumpus adventure game, developed as part of Oregon State University's CS162 course.

## Features

- Cave exploration
- Hazards and enemy encounters
- Player movement and decision-making
- Object-oriented design
- Polymorphism

## Skills Demonstrated

- Python
- Object-Oriented Programming
- Inheritance and Polymorphism
- Game Logic
- Program Organization

## Running the Project

1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Run the main Python file.  -> python main.py
